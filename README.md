# diesel-limits

**Оперативная карта ограничений и цен на дизельное топливо для перевозчиков**
по регионам России. Обновляется раз в сутки. Целевая аудитория — грузовые
перевозчики, которым нужно знать, где можно заправить грузовик и какие лимиты.

## Что показывает карта

5 уровней жёсткости ограничений для грузового транспорта:

| Уровень | Цвет | Описание |
|---------|------|----------|
| 0 — Свободно | синий | нет ограничений |
| 1 — Мягкие лимиты | зелёный | 100-200 л (хватит для бака грузовика) |
| 2 — Средние лимиты | жёлтый | 40-99 л (не хватит для полного бака) |
| 3 — Жёсткие лимиты | красный | < 40 л, только в бак, запрет канистр |
| 4 — Дефицит / ЧС | бордовый | АЗС закрыты, режим ЧС |

По клику на регион — карточка с:
- ценой ДТ (руб/л) и изменением за неделю
- спарклайном цены за 30 дней (тренд, min/max/среднее)
- текстом ограничений
- блоком «Для грузовиков»: лимиты по сетям АЗС (Газпромнефть, Лукойл, Роснефть, Татнефть и др.)
- блоком «История изменений»: таймлайн изменений лимитов (ужесточение ▼ / послабление ▲ / новое)

В шапке — строка управления:
- **Поиск региона** — с автодополнением, зумит карту к региону и открывает карточку
- **Переключатель Карта / Таблица** — таблица 85 регионов с сортировкой по колонкам
- **Фильтры** — по уровню ограничений и по сети АЗС; состояние сохраняется в URL

Справа — оперативные блоки:
- **Критические регионы** (уровни 3-4) — главная оперативная информация
- **Рост цен за неделю** — топ регионов по удорожанию
- **Легенда** — расшифровка цветов

## Структура репозитория

```
index.html              — фронтенд (чистый D3+SVG, без Leaflet)
dump_data_json.py       — SQLite → /srv/static/data.json (5 уровней + лимиты)
clean_geometry.py       — упрощает GeoJSON: только главные полигоны,
                          обработка 180-го меридиана, без островов
gen_diesel_map.py       — статичный PNG-хитмап для Telegram
publish_diesel_map.py   — отправка PNG в Telegram-канал
send_text.py            — отправка текста в Telegram-канал
```

## Технологии

- **Frontend**: D3.js 7.9.0 (CDN jsdelivr, с SRI) + чистый SVG. Без Leaflet.
- **Геометрия**: `russia_clean.geojson` (1 полигон на регион, без сотен островов)
- **Backend**: Python 3, SQLite
- **Telegram**: requests + Bot API

## Безопасная настройка

### 1. Telegram-бот

```bash
export TG_BOT_TOKEN='123456:ABC-DEF...'   # от @BotFather
export TG_CHAT_ID='-1001234567890'         # ID канала (опционально)
```

> ⚠️ Если старый токен когда-либо был в git-истории — отзовите его через
> @BotFather командой `/revoke`.

### 2. Запуск

```bash
# Обновить data.json (для сайта)
python3 dump_data_json.py

# Сгенерировать PNG-хитмап и отправить в Telegram
python3 gen_diesel_map.py
python3 publish_diesel_map.py

# Отправить текст в канал
python3 send_text.py "Сегодня обновлены лимиты в 5 регионах"
```

### 3. Веб-страница

`index.html` автономный — данные встроены в него. Для использования с
серверными данными замените блок `const FEATURES = [...]` на `fetch('/static/data.json')`.

Минимальная конфигурация nginx:

```nginx
location /static/ {
    alias /srv/static/;
    add_header Cache-Control "no-cache";
}
location / {
    root /srv/site;
    index index.html;
}
```
